# Icons

Vega Components use **Lucide Icons** by default.

### Example (Lucide)

To use an icon, import it from `lucide-react`:

<div style={{ display: "flex", gap: "10px", alignItems: "center" }}>
  <ArrowDown color="#3b82f6" size={24} />
  <Info color="#6b7280" size={24} />
</div>

```jsx
<div style={{ display: "flex", gap: "10px", alignItems: "center" }}>
  <ArrowDown color="#3b82f6" size={24} />
  <Info color="#6b7280" size={24} />
</div>
```

### Example (Icon Font)

If you prefer using an icon font, use the `className` attribute:

```jsx
<div>
  <i className="i icon-a-arrow-down" style={{ fontSize: "24px" }}></i>
</div>
```
